❌ TROUBLESHOOTING:

Problem: "Python not found"
Solution: Install Python and make sure "Add to PATH" was checked

Problem: "pip install fails"
Solution: Run as administrator (Windows) or use sudo (Mac/Linux)

Problem: "Bridge can't connect"
Solution: Check your internet connection - the cloud server might be down

Problem: "Lua script not responding"
Solution: In Reaper, go to Actions → Show action list → 
         Find "reaper_cloud_agent.lua" → Select it → Click "Run"

Problem: "Commands not working"
Solution: Make sure all 3 are running:
         1. Reaper with Lua script
         2. Bridge (START_BRIDGE.bat/sh)
         3. Browser with web interface open
